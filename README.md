# JEE Scholar Planner (PWA) 📚🚀

A high-performance, responsive Progressive Web App (PWA) engineered specifically for JEE Main & Advanced aspirants. This application serves as an intelligent study companion, providing a structured approach to master the vast JEE syllabus through a distraction-free, local-first interactive environment.

---

## 🌟 Key Features

### 📦 1. Progressive Web App (PWA) Architecture
- **Offline First:** Access your syllabus checklist, study schedule, and history without an internet connection via service workers.
- **Installable Native Feel:** Seamlessly installs on Android, iOS, and Desktop directly from the browser with zero bloat.
- **Persistent Local Storage:** All configuration states, checklists, planner schedules, and mock metrics are safely handled using `LocalStorage` or `IndexedDB`.

### 📊 2. Unified Performance Dashboard
- **Daily Focus Engine:** Displays immediate custom daily assignments ("Today's Target") alongside an intuitive click-to-complete resolution button.
- **Syllabus Progress Rings:** Vibrant, multi-axis radial trackers calculating precise syllabus completion rates for Physics, Chemistry, and Mathematics independently.
- **Consistency & Streak Engine:** Automatically logs contiguous target completions to compound daily momentum and accountability.

### 🗺️ 3. Multi-Tier Interactive Syllabus Checklist
- **Hierarchical Layout:** Navigable accordion infrastructure organized across **Subject → Core Conceptual Unit → Chapter**.
- **Granular Status Metrics:** Track chapter execution metrics across three dedicated vectors:
  - 📖 *Theory Accomplished*
  - 📝 *Problem-Solving/MCQ Targets Reached*
  - 🔁 *Periodic Revision Completed*
- **Instant Filtering:** Real-time search indexing for immediate retrieval of specific chapters or topics across the entire codebase.

### 🗓️ 4. Dynamic Study Planner & Backlog Handler
- **Agile Calendar Mapping:** Intuitive weekly or monthly grids for scheduling chapter targets and revisions.
- **Backlog Isolation System:** Missed checkpoints are systematically isolated into a dedicated red-flagged "Backlog Ledger", preventing layout drift until rescheduled.

### 📈 5. Deep Mock Test Analytics
- **Granular Result Auditing:** Custom modal input fields charting parameters across Total Aggregate Score, Subject-Specific Subtotals, and Cumulative Negative Marking impact.
- **Chronological Data Visualization:** Line graph analytics demonstrating long-term trend trajectories and score variances.
- **Categorical Error Analysis:** Self-reported performance logs isolating systemic weaknesses (e.g., *Conceptual Misunderstandings*, *Arithmetical/Silly Faults*, *Time Management Flaws*).

## 🎨 Design Language & UI Spec

- **Aesthetic Direction:** Pro dark-mode interface optimized to counteract optical fatigue over extended nighttime revision shifts.
- **Palette Architecture:** Deep Slate and Dark Indigo baseline foundations mixed with high-contrast functional highlights:
  - ⚡ **Physics:** Neon Blue Accent
  - 🧪 **Chemistry:** Emerald Green Accent
  - 📐 **Mathematics:** Deep Orchid/Purple Accent
- **Responsive Paradigm:** Mobile-optimized physical interface anchored around an ergonomic bottom navigation layout designed for unified thumb operation.
